Detailed Description

Cette classe gère le défilement de leds NeoPixels. Une seule couleur à la fois. Un motif de luminosité peut être appliqué, c'est un tableau de diviseurs de la luminosité des leds. la taille du tableau détermine le nombre de leds allumées(max 32)
